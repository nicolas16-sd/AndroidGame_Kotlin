Projeto desenvolvido em **Kotlin** utilizando **Jetpack Compose**, simulando a interface de um jogo estilo arcade.

## 🛠️ Tecnologias Utilizadas

- Kotlin
- Jetpack Compose
- Android Studio
- Material 3

---

## 🧠 Conceitos Trabalhados

- Layout com `Row`, `Column` e `Box`
- Controle de alinhamento (`Alignment` e `Arrangement`)
- Uso avançado de `Modifier`
- Organização de HUD (Head Up Display)
- Estruturação de telas com Composables
